# 🔥 Ride Streak Tracker – Mobile App UI/UX Design

A modern **Ride Streak Tracker** mobile application designed in **Figma**. This concept helps users build daily riding habits by tracking streaks, earning coins, monitoring progress, and staying motivated through gamification.

The interface follows a clean, minimal, and engaging design with intuitive navigation and reward-based interactions.

---

## 📖 Project Overview

Ride Streak Tracker is a gamified mobile app concept that encourages users to maintain daily riding consistency. Users can track their streaks, earn virtual coins, recover missed streaks, view statistics, and monitor progress through an easy-to-use dashboard.

This project demonstrates modern mobile UI principles with a focus on usability, motivation, and visual consistency.
